1. Download and install GitHub lib in local laptop
2. Create a GitHub repository: repo = frankfeng210/frank-mlops-cicd-project-3
3. Change to a local directory: dir = C:...\project3
4. Create a connection to GitHub and clone GitHub repo to local laptop. cmd:
 	1. gh repo clone frankfeng210/frank-mlops-cicd-project-3
5. Create structure for project 3
 	1. git status  							- check files
 	2. git add .   							- add files to staging area
 	3. git commit -m "copy project 3 data and python code files"
 	4. git push    							- push all files from staging area to GitHub repo
